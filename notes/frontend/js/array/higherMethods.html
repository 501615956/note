# forEach()

数组每个元素都执行一次回调函数(循环)。

语法：

arr.forEach((item, index, arr) => {
  console.log(item, index, arr)
})
参数：
- item: 数组中的每项元素
- index: 每项元素的做索引值
- arr: 数组本身

注意：forEach方法没有返回值，不能return，且该方法直接改变的是原数组。

例子：

let arr = [1, 2, 3, 4]
arr.filter(item => item + 1) 
console.log(arr) // [2, 3, 4, 5]

参数：
- item: 数组中的每项元素

# find()

  console.log(item, index, arr)
  return 判断条件
})
参数：
- item: 数组中的每项元素
- index: 每项元素的做索引值
- arr: 数组本身

  {name: '张三', age: 19},
  {name: '李四', age: 20}

# reduce()

将数组所有的元素计算为一个值（从左到右）

语法:

arr.reduce((prod, item) => {
  console.log(prod, item)
  return prod + item
}, value)
参数：
- prod: 之前计算的值
- item: 每项元素的做索引值
- value: 计算的初始值，默认为`0`
1

例子:

let arr = [1, 2, 3, 4]
let newValue = arr.reduce((prod, item) => {
  return prod + item
}, 0)
console.log(newValue) // 10

# every()

# map()

处理数组的每个元素，并返回处理后的数组。

语法:

arr.map((item, index, arr) => {
  console.log(item, index, arr)
  return 判断条件/对象
})
参数：
- item: 数组中的每项元素
- index: 每项元素的做索引值
- arr: 数组本身
1
2
3
4
5
6
7
8

例子:

let arr = [
  {name: '张三', age: 19},
  {name: '李四', age: 20}
]
// 用法一
let arr1 = arr.map(item => item.name) 
console.log(arr1)  // ['张三', '李四']
// 用法二
let arr2 = arr.map(item => {
  return {
    ...item,
    item.sex = '男'
  }
})
console.log(arr2) // [{name: '张三', age: 19, sex: '男'}, {name: '李四', age: 20, sex: '男'}]
