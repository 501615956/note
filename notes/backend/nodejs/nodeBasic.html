# 安装

尽管 NVM 并不支持 windows，但是可以运行在 WSL（Windows Subsystem for Linux）中，WSL 是一个可以运行在 Windows10上的一个 Linux子系统，WSL 安装包的地址：https://github.com/coreybutler/nvm-windows/releases (opens new window)

点击 nvm-setup.zip 即可下载安装

# 使用

# 安装指定node版本
nvm install v14.15.0
# 运行指定node版本
nvm use v14.15.0
# 切换到最新的node版本
nvm use node
# 给不同的版本号设置别名
nvm alias node_cms 14.15.0
# 使用该别名
nvm use node_cms
# 查看已安装node列表
nvm ls 或 nvm list

# 用于卸载某个node版本
nvm uninstall (version)

# 用于展示当前在windows系统运行的nvm版本
nvm version 或 nvm -v
