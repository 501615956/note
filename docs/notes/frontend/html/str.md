## HTML转义字符

| 显示结果 |   描述   |      实体名称       |
| :------: | :------: | :-----------------: |
|          |   空格   |      `&nbsp;`       |
|    <     |  小于号  |       `&lt;`        |
|    >     |  大于号  |       `&gt;`        |
|    &     |   和号   |       `&amp;`       |
|    "     |   引号   |      `&quot;`       |
|    '     |   撇号   | `&apos;` (IE不支持) |
|    ￥     |  人民币  |       `&yen;`       |
|    ©     |   版权   |      `&copy;`       |
|    ®     | 注册商标 |       `&reg;`       |
|    ×     |   乘号   |      `&times;`      |
|    ÷     |   除号   |     `&divide;`      |
